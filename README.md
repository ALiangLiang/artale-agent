# 🎹 Artale Agent - 楓之谷奧術傳說輔助計時器

這是一個專為楓之谷奧術傳說 (Artale) 設計的非侵入式按鍵計時器。它能幫助您精準掌控 Buff 剩餘時間，並提供視覺與音效的雙重提醒。

## ✨ 核心功能

- **多工並行計時**：支援多個按鍵同時倒數，自動根據剩餘時間從長到短排序（左至右）。
- **視覺化定位**：點擊「調整顯示位置」會出現黃色箭頭，隨拉即動，支援**跨螢幕（多螢幕）**拖放。
- **靈活熱鍵設定**：
    - 按下 `Pause Break` 鍵開啟設定介面。
    - 支援自定義熱鍵（F1-F12、字母、Alt/Shift/Ctrl 等）。
    - 內建精美職業分類圖示包（戰士、法師、弓箭手、盜賊、海盜、通用、道具）。
- **智慧提醒機制**：
    - **紅字預警**：倒數低於 30 秒自動變紅字。
    - **音效通知**：剩餘 20 秒（嗶 1 聲）、歸零時（嗶 2 聲）。
    - **歸零停留**：Buff 歸零後會以「紅字 0」停留 10 秒並持續每秒嗶一聲，確保您不會忘記補 Buff。
- **操作優化**：
    - **滑鼠右鍵取消**：對著計時圖示按右鍵可立即清除計時。
    - **智慧焦點鎖定**：只有在 `msw.exe` (Artale 遊戲視窗) 被選取時，按鍵才會觸發計時，避免在外部聊天或操作時誤觸。
- **非侵入式設計**：透明背景，不影響遊戲畫面，滑鼠可直接穿透計時器點擊遊戲。

## 🚀 快速開始

### 開發者執行
1. 安裝對應環境：`pip install PyQt6 pynput pywin32 psutil`
2. 啟動主程式：`python main.py`

### 建立執行檔 (EXE)
本專案內附自動化打包腳本：
1. 開啟 PowerShell 並進入專案目錄。
2. 執行 `.\build.ps1`
3. 打包完成後，可以在 `dist/ArtaleAgent.exe` 找到編譯好的執行檔。

## 🛠️ 操作說明
- **新增計時器**：在設定視窗按「新增按鍵」，然後直接按一下鍵盤上要綁定的按鍵即可。
- **更換圖示**：點擊按鍵旁的相框架圖示，即可進入分類資料夾選擇對應的技能圖示。
- **手動取消**：在遊戲中對著顯示的倒數圖示按**滑鼠右鍵**即可取消。
- **儲存設定**：所有變動均需按「儲存並套用」才會生效並寫入 `config.json`。

## ⚠️ 注意事項
- 本工具僅提供計時與提醒功能，**不包含自動按鍵、掛機等違反遊戲規範之機制**。
- 程式需在 Windows 環境下執行（使用了 `win32api` 與 `winsound`）。
- 打包後的 EXE 體積約 35MB，已內含所有必要的圖示資源。
