# 🍁 Artale Agent - Artale 全方位冒險助手

這是一個專為 MapleStory Worlds - Artale 設計的非侵入式遊戲輔助工具。它不僅提供精準的 Buff 計時提醒，更整合了高效率的經驗值監測系統與團隊路徑同步工具，助您在冒險中如虎添翼。

## ✨ 核心功能

### ⏲️ 智慧計時系統
- **多工倒數**：支援無限按鍵同時倒數，自動依剩餘時間排序（左至右）。
- **視覺與音效提醒**：
    - **紅字預警**：低於 30 秒自動變紅。
    - **節奏音效**：剩餘 20 秒（嗶 1 聲）、歸零時（嗶 2 聲）、歸零後停留 10 秒並每秒嗶聲確保補 Buff。
- **操作熱鍵**：
    - `Pause Break`：快速開啟控制中心。
    - `F12`：一鍵重置清空所有計時器 (可自定義)。
    - **右鍵取消**：滑鼠對準計時器圖示點擊右鍵即可立即清除。

### 📊 經驗值監測面板 (EXP Tracker)
- **高效率擷取**：採用 Windows Graphics Capture API，極低 CPU 占用。
- **實時 OCR 辨識**：自動追蹤遊戲畫面中的 EXP 數據與百分比。
- **效率分析**：
    - **10 分鐘速率**：精準計算每 10 分鐘獲取的 EXP 量與百分比。
    - **升級預估**：根據當前效率，自動推算到達下一等級所需的剩餘時間。
- **靈活控制**：支援 `F10` 開關面板、`F11` 暫停/恢復紀錄，支援手動拖曳位置。

### 🎮 羅茱 (RJPQ) YzY 團隊同步工具
- **雲端同步**：與 YzY 官方網頁版相容，可跨平台使用。
- **標記系統**：
    - 點擊標記：直接在遊戲畫面的面板上點擊區塊進行標記。
    - **熱鍵標記**：自定義 1~4 鍵（如 Numpad）快速標記當前列。
- **連線機制**：自建房間或加入隊友房間，支援 101-104 角色選色區分。

### ⚙️ 進階管理功能
- **多配置組切換**：提供 F1~F9 九組配置空間，雙擊可無縫切換（例如：戰士組、練功組、組隊組）。
- **跨螢幕支援**：完美支援多螢幕環境，計時器與面板位置可自由拖曳。

## 🚀 快速開始

### 環境需求
1. **Windows 10/11** (建議 2004 版本以上以支援新版捕捉 API)。
2. **Tesseract-OCR**：經驗值辨識核心，專案中已內附免安裝版。

### 開發者執行
1. 安裝 Python 套件：
   ```bash
   pip install PyQt6 pynput pywin32 psutil pytesseract opencv-python windows-capture PyQt6-WebSockets
   ```
2. 啟動主程式：
   ```bash
   python main.py
   ```

### 建立執行檔 (EXE)
本專案內附自動化打包腳本：
1. 以系統管理員權限開啟 PowerShell。
2. 執行 `.\build.ps1`。
3. 打包完成後，於 `dist/ArtaleAgent.exe` 執行。

## 🛠️ 操作說明
- **焦點保護**：按鍵監聽僅在 `msw.exe` (Artale 視窗) 置頂時觸發，聊天中不必擔心誤觸。
- **非侵入式**：背景完全透明，滑鼠可自由穿透計時器操作遊戲，不影響走位與點擊。
